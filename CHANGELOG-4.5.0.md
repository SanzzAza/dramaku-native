# Dramaku 4.5.0 — Visual Upgrade

Upgrade tampilan biar lebih “app premium”, tanpa ubah alur fitur.

- Palette lebih gelap-cinematic + aksen mint lebih hidup

- Card poster lebih dalam (gradient, badge blur, shadow)
- Spotlight / mood / top10 lebih premium
- Splash modern + loading bar
- Detail, search, settings, toast, empty state dipoles
- `perf-mode` tetap meredam animasi/shadow di HP kentang
